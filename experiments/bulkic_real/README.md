# Bulk IC Real Data

Performing a bulk IC inversion using real data from [Brett et al., 2024](https://www.nature.com/articles/s41561-024-01539-6).
Data available from [here](https://figshare.com/articles/dataset/Inner_Core_Travel_Time_Data_Reference_Model_PREM/26535394/1?file=48335947).

Inverting for nested P-wave Love parameters (A, C-A, F-A) and tilt angles (eta1, eta1).
